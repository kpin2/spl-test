The goal of this project is to provide players with a toolset that can be used to easily analyze and visualize data from Splinterlands, the biggest Play-to-Earn game on the Hive blockchain. All Splinter-Masters tools are provided free of charge, though tips are always appreciated! See the contact page for more information.

All the information found here is available publicly on the Hive blockchain and through the Splinterlands API, for which no official documentation exists. See the 'Resources' page to view our API documentation which uses custom endpoints to access the Splinterlands API.

Looking for more information on the Hive blockchain or Splinterlands? Check out our resources page.

On the Hive blockchain keeping track of your account address is simple. No long, complicated strings of numbers and letters to keep track of. Each blockchain address is just the username that was picked during account creation, preceded by the @ symbol. This in turn is also your Splinterlands username, and will be used across the entire Hive ecosystem. One of the few downsides to this type of setup being that you cannot change your username once the account has been created.

Using a Splinterlands username there are multiple categories of data that can be pulled from the blockchain, via various back-end APIs. Select one from the list below, enter a valid hive username (defaults to one of my accounts otherwise), and press search!
